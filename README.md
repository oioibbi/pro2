# 在线德州扑克

一个可直接部署到静态托管平台的德州扑克网页项目。

## 本地运行

直接打开 `index.html` 即可。

## 通过 GitHub Desktop 发布到 GitHub Pages

1. 在 GitHub Desktop 中选择 `File -> Add local repository...`
2. 选择这个文件夹：`G:\all\pro2`
3. 如果提示发布仓库，选择 `Publish repository`
4. 发布后打开 GitHub 网页仓库
5. 进入 `Settings -> Pages`
6. 在 `Build and deployment` 里选择：
   - `Source`: `Deploy from a branch`
   - `Branch`: `main`
   - `Folder`: `/ (root)`
7. 保存后等待几分钟，网站就会出现在 GitHub Pages 地址上

默认主页文件是 `index.html`，所以不需要额外构建。
